deutsche Einleitung

Falls Sie unter GNU/Linux arbeiten, sollten die benötigten Software-Pakete über die Paket-Verwaltung Ihrer Distribution bezogen werden. Sie werden in den meisten Distributionen vorhanden sein.

Notwendige Software

  • PostgreSQL, das Datenbank-Management-System, das von Zeugs für die Hauptdatenbank eingesetzt wird

Falls Sie nicht das Installationspaket verwenden, werden Sie auch folgende Software benötigen

  • Python, die Programmierspache in der das Zeugs-System geschrieben ist. Auch: Informationen auf Deutsch.
  • PyQt4, die graphische Bibliothek, auf der die Benutzeroberfläche basiert.
  • psycopg2, die Python-Schnittstelle zur PostgreSQL-Datenbank.
  • PyEnchant, die Python-Schnittstelle zur 'Enchant'-Rechtschreibprüfung.
  • Wörterbücher für die Rechtschreibprüfung. Einige sind im PyEnchant-Paket für Windows schon enthalten.

Nützliche Software

  • pgAdmin III, ein Verwaltungswerkzeug für PostgreSQL.
  • Acrobat reader, um PDF-Dateien anzuzeigen
  • OpenOffice - ein freies Office-Paket. Die Bedienungsanleitung wurde damit geschrieben.
  • The GIMP - freie Software zur Bildverarbeitung. Deutsche Dokumentation.
  • Inkscape - freie Software zur Verarbeitung von Vektor-Grafik-Bilder (SVG).
  • 7-Zip - freie Archivierungssoftware (kann z.B. '.zip'- und '.tar.gz'-Archive entpacken).

Für Entwickler: zur Erstellung des Windows-Installationspakets

  • PyInstaller. Die zur Zeit aktuelle Version, 1.3, ist nicht ohne Weiteres PyQt4-fähig. Man sollte die neueste Version vom 'Subversion Repository' herunterladen.
  • NSIS
